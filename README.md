#  INFORME LABORATORIO 1

## 1.- OBJETIVOS
## GENERAL 
- Analizar la ley de Kirchhoff mediante la construcción de un circuito eléctrico mixto para comprend
## ESPECIFI
- Analizar y explicar la ley de Kirchhoff para los voltajes como para la corriente.
